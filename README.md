Important:
- the playlist is saved in the user's browser on their device
- it does not sync between devices unless a backend is added later
- users upload their own MP3s locally in the browser

How to publish on GitHub Pages:
1. Create a new GitHub repository, for example `playdeck-uploader`
2. Upload everything inside this folder to the root of that repository
3. In GitHub, open `Settings` -> `Pages`
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Save it
6. Wait for GitHub Pages to publish the site

Your link will usually look like:
- `https://YOUR-USERNAME.github.io/REPO-NAME/`

If you want the cleanest GitHub Pages setup, use this folder exactly as the repository contents.
